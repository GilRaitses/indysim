\section*{Data and Code Availability}

All code and processed data are available at
\url{https://github.com/GilRaitses/indysim} (InDySim: Interface Dynamics Simulation Model). Data
preprocessing used magatfairy
(\url{https://github.com/GilRaitses/magatfairy}) for converting MAGAT
Analyzer MATLAB exports to HDF5 and retrovibez
(\url{https://github.com/GilRaitses/retrovibez}) for reverse crawl
detection.
